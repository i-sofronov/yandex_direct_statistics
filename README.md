

# Подключение Яндекс.Директ API
Занимался я как-то подключением API Директа и подумал: почему бы не залить простенькую систему получения и вывода статистики по кампаниям к себе в гит?


## Обзор Архитектуры

В приложение используется архитектуру, которая объединяет Laravel на бэкенде и Vue.js на фронтенде, интегрируясь через Inertia.js.

*  Бэкенд (Laravel)

*  Фронтенд (Vue.js с Inertia.js)

*  Разработка & Сборка: Vite

---

## Базовый функционал
- Добавление проекта
- Прикрепление аккаунта Директа к проекту
- Получение токена для работы с API
- Получение статистики и кампаний
- Вывод статистики за период
